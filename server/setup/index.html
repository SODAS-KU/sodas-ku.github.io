    Github

    Generate SSH key

    Follow the guide at GitHub.

    SSH Config

    Create a config file at ~/.ssh/config and add the following lines:

    Host github.com
    HostName ssh.github.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_ed25519
    Port 443
